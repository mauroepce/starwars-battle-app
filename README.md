## 💡 Introduction

⚔️ A player 1 vs player 2 concept game based on the Star Wars theme. The winner is chosen at random.

## 🚀 Install

```sh
$ git clone https://github.com/mauroepce/starwars-battle-app
$ npm i && npm run dev
```
